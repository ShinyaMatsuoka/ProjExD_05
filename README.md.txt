# こうかとん探検隊
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターこうかとんを矢印、スペースで操作を行い、
sキーで敵を倒していく。
敵がいなくなるとクリア
ライフがなくなるとゲーム終了

## ゲームの実装
###共通基本機能
* ブロックに関するクラス
* エネミーの移動や、攻撃されたときなどに関するクラス
* sキーを押したときにビームを出すクラス
* こうかとんの生成や移動、キーを押した時の行動などに関するクラス
* マップ生成に関するクラス
* ゲームの画像の読み込みなどに関するクラス

### 担当追加機能
* Shot_a:ビームが斜めに出るようになり、2回壁を跳ね返ることができる。

# こうかとん探検隊
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターこうかとんを矢印、スペース操作を行い、
sやaボタンで敵を倒していく。

## ゲームの実装
###共通基本機能
* ブロックに関するクラス
* エネミーに関するクラス
* キーsを押したときにビームを出すクラス
* こうかとんに関するクラス
* マップに関するクラス
* ゲームの画像などに関するクラス

### 担当追加機能
* Shot_a:ビームが斜めに出るようになり、2回壁を跳ね返ることができる。
